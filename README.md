Age Calculator
A simple Python script that asks for a date of birth in dd/mm/yyyy format and calculates age in multiple ways.

Features
Age in years
Age in days
Age in hours
Age in minutes
Age in seconds

How it works
The script:

Prompts the user to enter their date of birth.
Splits the input into day, month, and year.
Creates a Python date object.
Compares it with today's date.
Prints the calculated age values.

Run it
Make sure Python is installed, then run:
bash
python age_calculator.py
Example input

text
04/07/1996
Example output

text
You are 30 years old
You are 10,957 days old
You are 262,968 hours old
You are 15,778,080 minutes old
You are 946,684,800 seconds old
Notes
Input format should be dd/mm/yyyy

Invalid dates should be handled with validation if you extend the script
